  1. Run this server yourself. The source code is open source, which means you can edit the code, and have fun with it yourself! You can find the source code here.

How does it work?

I built this project mostly during the first three months of 2020, as part of a personal project. The town generator combines the town's name and the size into a "seed" to randomly generate all the information in the town. There's some more information theoretical optimalizations involved to speed up the process, but that's all.
